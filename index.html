Minefinder

Minefinder is a minesweeper clone and full stack web app.

Tech Stack: MySQL, Express, React, NodeJS, Material-UI, PassportJS, Font Awesome.

Features:

  • Fully functional minesweeper clone.
  • User statistics (average time, best time, win rate).
  • Leaderboards for each board size and each statistic.

See it live here. The app sleeps after 30 minutes of inactivity so it may take a moment for the page to load.
